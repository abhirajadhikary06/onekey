Onekey with OpenRouter

OpenRouter provides unified access to multiple AI models from different providers through a single API. Integrate Onekey to manage your OpenRouter API keys and access models from OpenAI, Anthropic, Google, Meta, and more with centralized monitoring.

Quick Start

import requests

msgs = [
    "What is machine learning?",
    "Explain Docker in simple terms",
    "What is the difference between REST and GraphQL?",
] * 2

for m in msgs:
    print(requests.post(
        "https://onekey-8pr2.onrender.com/proxy/u/openrouter/<api-key-name>",
        headers={
            "Authorization": "Bearer apikey-openrouter-<api-key-name>",
            "Content-Type": "application/json"
        },
        json={
            "model": "openai/gpt-4-turbo-preview",
            "messages": [{"role": "user", "content": m}]
        }
    ).json())
curl -X POST https://onekey-8pr2.onrender.com/proxy/u/openrouter/<api-key-name> \
  -H "Authorization: Bearer apikey-openrouter-<api-key-name>" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai/gpt-4-turbo-preview",
    "messages": [
      {
        "role": "user",
        "content": "What is dbt tool?"
      }
    ]
  }'

Parameters

  • <api-key-name> - Replace with your API key name from the dashboard
  • model - OpenRouter model ID (e.g., openai/gpt-4, anthropic/claude-3-opus)
  • messages - Array of message objects with role and content

Popular OpenRouter Models

  • openai/gpt-4-turbo-preview - OpenAI's GPT-4 Turbo model
  • anthropic/claude-3-opus - Anthropic's most capable Claude model
  • google/gemini-pro - Google's Gemini Pro model
  • meta-llama/llama-3-70b-instruct - Meta's Llama 3 70B instruction model
  • mistralai/mixtral-8x7b-instruct - Mistral's mixture of experts model
