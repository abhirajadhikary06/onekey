Onekey with Mistral AI

Mistral AI delivers state-of-the-art open-source language models with exceptional performance and efficiency. Their models like Mixtral 8x7B and Mistral 7B offer powerful reasoning capabilities while maintaining fast inference speeds. Use Onekey to seamlessly integrate Mistral AI models into your applications with unified API key management.

Quick Start

import requests

msgs = [
    "What is machine learning?",
    "Explain Docker in simple terms",
    "Write a Python function to reverse a string",
] * 2

for m in msgs:
    print(requests.post(
        "https://onekey-8pr2.onrender.com/proxy/u/mistral/<api-key-name>",
        headers={
            "Authorization": "Bearer apikey-mistral-<api-key-name>",
            "Content-Type": "application/json"
        },
        json={
            "model": "mistral/mistral-large-latest",
            "messages": [{"role": "user", "content": m}]
        }
    ).json())
curl -X POST https://onekey-8pr2.onrender.com/proxy/u/mistral/<api-key-name> \
  -H "Authorization: Bearer apikey-mistral-<api-key-name>" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "mistral/mistral-large-latest",
    "messages": [
      {
        "role": "user",
        "content": "Explain the concept of neural networks in simple terms"
      }
    ]
  }'

Parameters

  • <api-key-name> - Replace with your API key name from the dashboard
  • model - Mistral model ID (e.g., mistral/mistral-large-latest)
  • messages - Array of message objects with role and content
  • temperature - Sampling temperature (0.0 to 1.0, default: 0.7)
  • max_tokens - Maximum tokens to generate (optional)
  • top_p - Nucleus sampling parameter (0.0 to 1.0, default: 1.0)

Popular Mistral AI Models

  • mistral/mistral-large-latest - Latest and most powerful Mistral model
  • mistral/mistral-medium-latest - Balanced performance and cost
  • mistral/mistral-small-latest - Fast and cost-effective for simple tasks
  • mistral/open-mistral-7b - Open-source 7B parameter model
  • mistral/open-mixtral-8x7b - Open-source mixture of experts model
  • mistral/open-mixtral-8x22b - Open-source 8x22B mixture of experts model
  • mistral/codestral-latest - Specialized for code generation and understanding

Example: Code Generation

import requests

response = requests.post(
    "https://onekey-8pr2.onrender.com/proxy/u/mistral/<api-key-name>",
    headers={
        "Authorization": "Bearer apikey-mistral-<api-key-name>",
        "Content-Type": "application/json"
    },
    json={
        "model": "mistral/codestral-latest",
        "messages": [
            {
                "role": "user",
                "content": "Write a Python function that implements binary search on a sorted array"
            }
        ],
        "max_tokens": 500
    }
)

print(response.json())

  • Fast Inference - Optimized for low-latency, high-throughput applications
  • Strong Reasoning - Excellent performance on complex reasoning tasks
  • Code Specialization - Codestral model specifically trained for programming tasks
  • Cost-Effective - Competitive pricing compared to other leading models
