Onekey with Perplexity

Perplexity AI provides research-focused language models with built-in web search and citation capabilities. Their models excel at providing accurate, up-to-date information with verifiable sources. Use Onekey to seamlessly integrate Perplexity AI into your applications with unified API key management.

Quick Start

import requests

msgs = [
    "What is machine learning?",
    "Explain Docker in simple terms",
    "What are the latest AI trends in 2025?",
] * 2

for m in msgs:
    print(requests.post(
        "https://onekey-8pr2.onrender.com/proxy/u/perplexity/<api-key-name>",
        headers={
            "Authorization": "Bearer apikey-perplexity-<api-key-name>",
            "Content-Type": "application/json"
        },
        json={
            "model": "perplexity/llama-3.1-sonar-small-128k-online",
            "messages": [{"role": "user", "content": m}]
        }
    ).json())
curl -X POST https://onekey-8pr2.onrender.com/proxy/u/perplexity/<api-key-name> \
  -H "Authorization: Bearer apikey-perplexity-<api-key-name>" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "perplexity/llama-3.1-sonar-small-128k-online",
    "messages": [
      {
        "role": "user",
        "content": "What are the key differences between GPT-4 and Claude 3?"
      }
    ]
  }'

Parameters

  • <api-key-name> - Replace with your API key name from the dashboard
  • model - Perplexity model ID (e.g., perplexity/llama-3.1-sonar-small-128k-online)
  • messages - Array of message objects with role and content
  • temperature - Sampling temperature (0.0 to 1.0, default: 0.7)
  • max_tokens - Maximum tokens to generate (optional)
  • return_citations - Include citations in response (true/false, default: false)
  • return_images - Include images in response (true/false, default: false)

Popular Perplexity Models

  • perplexity/llama-3.1-sonar-small-128k-online - Fast online model with 128K context
  • perplexity/llama-3.1-sonar-large-128k-online - Powerful online model with 128K context
  • perplexity/llama-3.1-sonar-small-128k-chat - Fast offline chat model
  • perplexity/llama-3.1-sonar-large-128k-chat - Powerful offline chat model
  • perplexity/llama-3.1-70b-instruct - Large offline instruction model
  • perplexity/llama-3.1-8b-instruct - Smaller offline instruction model

Online vs Offline Models

  • Online models (e.g., -online suffix) - Access real-time web search with citations
  • Offline models (e.g., -chat suffix) - Faster, no web search, uses training data only
  • Use online when you need current information with sources
  • Use offline when you need faster responses or don't need web search

Example: Research with Citations

import requests

response = requests.post(
    "https://onekey-8pr2.onrender.com/proxy/u/perplexity/<api-key-name>",
    headers={
        "Authorization": "Bearer apikey-perplexity-<api-key-name>",
        "Content-Type": "application/json"
    },
    json={
        "model": "perplexity/llama-3.1-sonar-large-128k-online",
        "messages": [
            {
                "role": "user",
                "content": "What are the latest breakthroughs in quantum computing in 2025?"
            }
        ],
        "return_citations": True,
        "return_images": False
    }
)

result = response.json()
print(result)

# The response will include citations from web sources
# Check result['citations'] for source information

Key Features of Perplexity AI

  • Real-time Web Search - Access up-to-date information from the internet
  • Automatic Citations - Get sources for all factual claims
  • Large Context Window - 128K tokens for processing long documents
  • Research-Focused - Optimized for accurate, verifiable answers
  • Fast Inference - Low latency even with web search enabled
  • Flexible Models - Choose between online and offline modes
